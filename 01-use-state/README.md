# 📝 Card de perfil red social

Una aplicación que ayuda a gestionar una card de perfil de alguna red social.

## 🚀 Características

- ✅ Gestiona el follow del usuario.
- ✅ Muestra información relevante del perfil
- 🎨 Interfaz simple e intuitiva
